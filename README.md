# Twitter Sentiment Analysis (Team Project)

This project analyzes public sentiment from Twitter data using Natural Language Processing (NLP) and Machine Learning.

## Tools & Technologies
- Python
- Pandas, NumPy
- NLTK
- Scikit-learn
- TF-IDF
- Logistic Regression

## Dataset
Twitter US Airline Sentiment dataset (Kaggle)

## Steps to Run
1. Download the repository
2. Keep `Tweets.csv` in the same folder as the notebook
3. Open the notebook
4. Restart kernel and Run All cells

## Team Project
This project was developed as part of an academic/industrial training assignment.
